CSDtv Ad Controller — Setup Guide

Everything you need to install the commercial playout controller on a Mac and run it inside OBS. Follow the steps in order. Takes about 15 minutes.

Download the controller

⬇ Download csdtv-obs-controller.zip

A small app that plays commercials into OBS. It runs on the streaming Mac and you drive it from a panel docked inside OBS.

What you need

  • The streaming Mac with admin rights
  • OBS 28 or newer (WebSocket is built in)
  • Node.js 18 or newer — you'll install this in step 1
  • Your commercial video and image files

1Install Node

The controller runs on Node. Install it once, system-wide.

  1. Go to nodejs.org.
  2. Download the macOS Installer (.pkg). It picks the right version for your chip automatically.
  3. Run the installer and click through to Finish.
  4. Open Terminal (Applications → Utilities → Terminal) and type node --version, then press Return. You should see a version like v22.x.
Don't use nvm. If someone suggests "nvm," skip it. nvm is a version manager that only works inside a Terminal and won't be found by a double-click launcher. Use the plain .pkg installer above.

2Unzip the controller

  1. Double-click the downloaded csdtv-obs-controller.zip to unzip it.
  2. Move the resulting csdtv-obs-controller folder somewhere permanent, like your Documents folder. Don't leave it in Downloads.

Inside you'll find a launcher named Start Ad Controller.command, a config.json settings file, an ads folder for commercials, and a preshow folder for a "starting soon" video.

3Turn on the OBS WebSocket

This lets the controller talk to OBS.

  1. In OBS, go to Tools → WebSocket Server Settings.
  2. Check Enable WebSocket Server.
  3. Leave Server Port at 4455.
  4. Check Enable Authentication and set the Server Password.
  5. Click Apply, then OK.
Password: Use the CSDtv OBS password. Ask Justin for the password and enter the exact same value here and in step 5. They have to match.

4Add the OBS sources

In the scene that airs your show, add three sources. Names must match exactly, including capital letters.

Source typeName it exactlyWhat it's for
Media SourceCommercialsVideo ads
Image SourceCommercialsImageImage ads
Media SourceStartingSoonThe "starting soon" pre-show loop
  • Leave the Local File empty on all three. The controller fills them in.
  • Size and position them where ads should appear on screen.
  • Put StartingSoon below the two Commercials sources in the source list, so ads cover it when they play.
Scene name matters. Open config.json and make sure "scene" is set to the exact name of the scene holding these sources. It ships as "Live". If your scene has a different name, ads will load but stay hidden.

5Set the password in the config

  1. In the controller folder, right-click config.json → Open With → TextEdit.
  2. Find the line "password": "" near the top.
  3. Put the OBS password between the quotes, so it reads "password": "THEPASSWORD".
  4. Save and close.
Same password as step 3. Ask Justin if you don't have it. The value here must exactly match the OBS Server Password.

6Start the controller

  1. Make sure OBS is running.
  2. Double-click Start Ad Controller.command.
  3. First time only: macOS may say it's from an unidentified developer. Right-click the file → Open → Open. After that a normal double-click works.
  4. A small window opens. The first run installs what it needs (about 30 seconds), then starts.
  5. You'll see Connected to OBS when it's talking to OBS. Leave this window open during the show. Closing it stops the controller.

7Add the control docks in OBS

Docks are just web panels inside OBS. Add them once.

Ad Control dock

  1. In OBS, go to Docks → Custom Browser Docks.
  2. Dock Name: Ad Control
  3. URL: http://127.0.0.1:4466
  4. Click Apply. Drag the panel wherever you like.

Parade Graphics docks (if this Mac also runs parade graphics)

Same steps, add these two:

Dock NameURL
Parade Controlhttps://www.csdtvstaff.org/parade-control.html

And add a Browser source (not a dock) named Parade Graphics at https://www.csdtvstaff.org/parade-output.html, sized 1920 × 1080, placed above your camera.

8Get your commercials onto the Mac

  1. Download the commercial files from the CSDtv hub commercials library.
  2. Put video and image ads in the controller's ads folder.
  3. Put your "starting soon" hold video in the preshow folder.
  4. In the Ad Control panel, click Reload folder to pick them up.

Running the show

Two ways to air ads, both from the Ad Control panel:

  • Click an ad to air it. Every file in the ads folder is a button. Click one and it plays right away.
  • Starting Soon pre-show. Click Start pre-show. Your starting-soon video loops and ads drop in over it every 1–2 minutes. Click Stop pre-show to end it.

Other buttons:

  • ▶ Play random ad — fire a random ad immediately.
  • ■ Stop ad — pull whatever ad is on air right now.
  • Auto rotation — a plain random loop on the normal 3–10 minute gap.
  • Reload folder — pick up new files you dropped in without restarting.

Troubleshooting

The launcher says Node isn't installed, but I installed it

Open Terminal and run which node. If it prints nothing, Node didn't finish installing — redo step 1 with the .pkg installer and open a fresh Terminal. If it prints a path, close the launcher window and double-click Start Ad Controller.command again.

macOS won't open the launcher

Right-click Start Ad Controller.command → Open → Open. You only need to do this the first time.

The panel says "OBS offline"

  • Make sure OBS is running.
  • Check Enable WebSocket Server is on (step 3).
  • Confirm the port is 4455.
  • Confirm the password in config.json exactly matches the OBS Server Password. This is the most common cause. Ask Justin to confirm the password.

It says "Connected to OBS" but no ad appears

  • Check the source names are exactly Commercials, CommercialsImage, StartingSoon.
  • Check "scene" in config.json matches your real scene name.
  • Make sure the sources are in that scene and not hidden.

An image ad shows blank

Confirm you added the CommercialsImage Image Source in step 4.

A commercial won't show up in the list

Make sure the file is in the ads folder and click Reload folder. Videos should be .mp4 or .mov. Images should be .png or .jpg.

Starting Soon won't start

Put a video in the preshow folder and click Reload folder. The panel will say when a starting-soon video is ready.

The panel and OBS look out of sync

They resync within a second. If not, right-click the browser source in OBS → Refresh.
