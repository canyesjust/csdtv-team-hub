Canyons Innovation Center signage — visual reference

This is the look Cursor should match. Screen layouts use the navy palette and render full-screen in AbleSign. Admin pages use the Hub light or dark theme. Port this markup into the React components: keep the structure, palette, spacing, and font sizes, and swap the placeholder content for live data. See signage-design-spec.md for tokens and Cursor prompts.
